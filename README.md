# 江西财经大学自动评教

适用于江西财经大学 KINGOSOFT 教务系统的自动评教油猴脚本。

## 功能

- **浮动控制面板**：页面右下角常驻，可拖拽移动
- **自动评教**：自动选择最佳选项、填写意见、暂存结果
- **双模式支持**：快速模式（连续评教）/ 兼容模式（每次刷新）
- **自动降级**：快速模式连续失败 3 次自动切换兼容模式
- **清除评教**：一键删除所有暂存数据
- **注意事项自动处理**：自动检测并点击「我已阅读」
- **会话过期检测**：自动等待用户重新登录后继续
- **暂停/停止**：评教和清除过程中均可暂停或停止

## 使用方法

### 第一步：打开教务系统

登录 `https://jwxt.jxufe.edu.cn`，进入主页后，页面右下角会出现**自动评教浮动面板**。

### 第二步：选择模式

| 模式 | 说明 | 适用场景 |
| :--- | :--- | :--- |
| **快速模式**（默认） | 连续评教，不刷新页面，速度更快 | 网络稳定、系统状态正常时 |
| **兼容模式** | 每次评教后刷新页面，重置系统状态 | 快速模式报错或系统异常时 |

点击面板上的「快速」或「兼容」按钮切换。如果快速模式连续失败 3 次，脚本会自动切换到兼容模式。

### 第三步：开始评教

1. 确保已登录教务系统
2. 点击面板上的「**开始**」按钮
3. 脚本会自动：
   - 导航到评教页面（如尚未在评教页面）
   - 自动处理「我已阅读」注意事项弹窗
   - 逐个点击「评价」→ 自动填写表单 → 暂存
4. 面板实时显示当前进度（如 `[5 剩余] 课程名`）
5. 全部完成后显示「全部完成」

### 第四步：清除评教数据（可选）

如果需要重置所有评教状态：

1. 先手动导航到评教页面
2. 点击面板上的「**清除评教**」按钮
3. 脚本会自动逐个点击「删除暂存」，并等待页面刷新
4. 清除过程中可随时点击「停止」中断

### 操作按钮说明

| 按钮 | 功能 |
| :--- | :--- |
| **开始** | 启动自动评教流程 |
| **暂停/继续** | 暂停或恢复评教/清除操作 |
| **停止** | 终止当前操作 |
| **快速/兼容** | 切换评教模式 |
| **清除评教** | 一键删除所有暂存的评教数据 |

## 工作原理

脚本采用逐个评教策略：

1. 遍历评教列表，找到第一个未评教的课程
2. 点击「评价」按钮打开评教表单
3. 自动选择所有选择题为最佳选项（ID 含 `_1` 的选项）
4. 自动填写所有意见框为「很好」
5. 点击「暂存」按钮保存
6. 关闭对话框，继续下一个
7. 快速模式：直接处理下一个（不刷新）
8. 兼容模式：刷新页面，重新导航后继续

## 注意事项弹窗机制

评教系统在用户点击「评价」后，会先弹出「注意事项」对话框（`dialog-frame` iframe）：

- 内置 10 秒倒计时，倒计时期间「我已阅读」按钮禁用
- 倒计时结束后按钮变为可点击状态
- 脚本通过递归搜索所有 `iframe`/`frame` 元素查找 `#btnClose` 按钮，检测到后自动点击

## 评教列表页面结构

评教列表（`DataTable.jsp`）嵌套在 `frmDesk` → `frame_1` → `frmReport` 三层 iframe 中：

| 列名 | DOM ID 模式 | 说明 |
|------|------------|------|
| 课程/环节 | `trN_kc` | 课程名称 |
| 教师 | `trN_js` | 教师姓名 |
| 评价得分 | `trN_pjdf` | 已评教时显示分数，未评教时为空 |
| 操作 | `trN_wjdc` | 含「评价」和「删除暂存」链接 |

脚本通过检查 `_pjdf` 列是否为空判断未评教状态，遍历「评价」链接确认可评教项。
