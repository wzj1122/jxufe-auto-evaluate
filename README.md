# 江西财经大学自动评教

适用于江西财经大学 KINGOSOFT 教务系统的自动评教油猴脚本。

## 功能

- 自动导航到评教页面
- 自动点击"我已阅读"注意事项
- 自动选择所有选择题为最高选项
- 自动填写所有意见框
- 自动暂存评教结果
- 每次评教后自动刷新页面，确保 KINGOSOFT 系统状态正常
- 自动处理会话过期，等待用户重新登录后继续
- 页面右上角实时显示评教进度

## 安装

1. 安装 [Tampermonkey](https://www.tampermonkey.net/) 浏览器扩展
2. 点击 Tampermonkey 图标 → 添加新脚本
3. 将 `auto-evaluate.user.js` 的内容粘贴进去
4. 保存（Ctrl+S）
5. 打开 `https://jwxt.jxufe.edu.cn/frame/homes.action*`
6. 点击 Tampermonkey 图标 → 江西财经大学自动评教 → **▶️ 开始评教**

## 使用

- **▶️ 开始评教**：开始自动评教流程
- **⏹️ 停止**：停止自动评教

## 工作原理

脚本采用"每次评教一个，刷新页面"的策略：

1. 点击第一个未完成评教的"评价"按钮
2. 等待评教表单加载
3. 自动选择所有选择题为最佳选项
4. 自动填写所有意见框为"很好"
5. 点击暂存按钮保存
6. 刷新页面，重置 KINGOSOFT 系统状态
7. 重复以上步骤直到所有评教完成

这种策略虽然每次评教之间需要刷新页面，但能确保 KINGOSOFT 的 dialog 系统状态正常，避免连续评教时出现的各种问题。
